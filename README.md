## Features

Provides highlighting for most keywords. I've also tried to add the formatting for nested conditionals or loops

## Installation

### Method 1: Download from GitHub Releases (Recommended)

1. Go to the [Releases page](https://github.com/Jeff-Barlow-Spady/sdev-pseudocode/releases)
2. Download the latest `sdev-x.x.x.vsix` file
3. Open VSCode
4. Go to Extensions (Ctrl+Shift+X or Cmd+Shift+X on Mac)
5. Click the `...` menu at the top of the Extensions panel
6. Select **"Install from VSIX..."**
7. Choose the downloaded `.vsix` file
8. Reload VSCode when prompted

### Method 2: Command Line Installation
```bash
code --install-extension sdev-0.0.1.vsix
```

### Method 3: Manual Installation

1. Download the `.vsix` file
2. Open VSCode
3. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
4. Type "Install from VSIX"
5. Select the downloaded file

## Usage

Once installed, simply open any file with `.sdev` or `.pcode` extension and enjoy syntax highlighting!

## Updating

To update to a newer version:
1. Download the latest `.vsix` file
2. Install it using any of the methods above
3. VSCode will automatically replace the old version

## Using

1. Create a file with the .sdev extension
2. Have fun
